# HR Portal (React)

A modern HR management portal built with React.

## Features

- Role-based authentication
- HR analytics dashboard
- Employee management system
- Leave request workflow
- Onboarding tracking
- Employee search and filtering
- Dark mode UI
- Responsive SaaS-style dashboard

## Technologies

- React
- React Router
- Recharts
- LocalStorage
- CSS

## Installation

npm install  
npm run dev
